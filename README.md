### 赛博数码管
+ 模拟数码管：字母&数字
+ 模拟4x4像素屏：符号
+ 一定程度上防烧屏：每次显示后会移动一个像素
+ 复古扫屏动画（参考旧电子秤的启动画面）
+ 顺序切换动画（从当前显示的数字开始逐个切换数字直至到达目标数字）
